RESTART THE GAME IF IT DOES NOT START IN FULLSCREEN
Controls:
Movement - WASD
Attack - LClick ( must have weapon equipped )
F - Interact
F/Space - Enter building
Exit/Space - Exit Building

C - Stats Menu
V - Abilities Menu
Q - Quest Menu
I - Inventory Menu 
B - recall to town
CTRL - Change weapon ( seen in lower-left corner )

CHEATS:
Open inventory ( I ) and press R to receive a set of items. Right Click on it to equip all. This allows instant combat actions
Open Abilities and upgrade them as points are available from the start.
